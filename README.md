# Frappe Bench Manager

A terminal UI to manage Frappe bench sites and databases.

## Features
- View all sites across all benches
- Detect orphaned databases (wasted space)
- Drop sites or orphaned DBs with confirmation

## Install

```bash
pipx install git+https://github.com/Sudhanshu-Badole/frappe-bench-manager
```

## Run

```bash
frappe-manager
```
